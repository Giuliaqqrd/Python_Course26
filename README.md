## Link utili per installazione degli strumenti fondamentali

### Package manager
Prima di installare è buona norma controllare se sia già presente un'installazione sulla vostra macchina:
- **Windows:** *WINGET*
  
Nel terminale digitare ``` winget ``` e controllare se l'output restituisce una versione esistente.
Se non è già installato, scaricare da Microsoft Store. Comandi e altre info: [Winget - info](https://learn.microsoft.com/en-us/windows/package-manager/winget/), [Winget - istruzioni](https://learn.microsoft.com/it-it/training/modules/explore-windows-package-manager-tool/3-install-uninstall-software) 

Una volta completata l'installazione è possibile controllare il percorso di installazione con ```where winget```

- **MacOS:** *HOMEBREW*

Nel terminale digitare ```brew --version```. Se non è già installato seguire [Homebrew - installazione](https://brew.sh/). 
Una volta completata l'installazione è possibile controllare il percorso di installazione con ```which brew```

**N.B:** Seguire bene le istruzioni da terminale.
