## Struttura del progetto

La struttura del progetto seguirà l’organizzazione tipica dei progetti di **Machine Learning**.  
Dopo aver scelto un problema o un tema di interesse, il progetto dovrà affrontare **almeno due** delle seguenti fasi:

1. **EDA (Exploratory Data Analysis)**  
   Analisi esplorativa del dataset per comprenderne la struttura, la qualità dei dati e le relazioni tra le variabili prima dell’addestramento dei modelli di machine learning.

2. **Preprocessing dei dati**  
   Preparazione e trasformazione dei dati per renderli adatti all’addestramento dei modelli (ad esempio gestione dei valori mancanti, normalizzazione o selezione delle feature).

3. **Addestramento dei modelli**  
   Applicazione di uno o più algoritmi di machine learning per affrontare il problema scelto.

4. **Valutazione dei risultati**  
   Analisi delle prestazioni dei modelli tramite metriche appropriate e interpretazione dei risultati ottenuti.

---

## Possibili combinazioni di attività

### 1. EDA + Preprocessing dei dati
- Analisi esplorativa del dataset per comprenderne struttura, distribuzione delle variabili e qualità dei dati.
- Sulla base dei risultati dell’EDA, applicazione di tecniche di preprocessing (es. gestione dei valori mancanti, selezione delle feature, normalizzazione).

Questa combinazione è utile quando l'obiettivo è **comprendere a fondo il dataset e prepararlo correttamente per futuri modelli**.

---

### 2. Preprocessing dei dati + Addestramento dei modelli
- Preparazione dei dati (pulizia, encoding di variabili categoriche, scaling, ecc.).
- Addestramento di uno o più modelli di machine learning sul dataset preprocessato.

Questa combinazione è utile quando si vuole **mostrare l’impatto della preparazione dei dati sulle prestazioni dei modelli**.

---

### 3. Addestramento dei modelli + Valutazione dei risultati
- Addestramento di uno o più modelli.
- Confronto delle prestazioni tramite metriche appropriate (accuracy, precision, recall, F1-score, ecc.).

Questa combinazione permette di **confrontare diversi approcci di machine learning**.

---

### 4. EDA + Addestramento dei modelli
- Analisi esplorativa del dataset per identificare pattern o relazioni tra le variabili.
- Addestramento di modelli di machine learning utilizzando le informazioni emerse dall’EDA.

Questa combinazione è utile per **motivare le scelte modellistiche sulla base dell’analisi dei dati**.

## Esempio

Si può scegliere di approfondire il punto **1 (EDA)** effettuando un’analisi esplorativa approfondita del dataset e successivamente proporre possibili strategie di preprocessing.

Ad esempio, nel caso di un problema di **classificazione di recensioni**, dopo aver analizzato il dataset si potrebbe:

- selezionare solo alcune **feature rilevanti**;
- applicare tecniche di **normalizzazione o scaling dei dati**;
- rimuovere o trasformare variabili meno informative.
